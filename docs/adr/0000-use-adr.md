# 0. Record Architecture Decisions

Date: 2026-01-04

## Status

Accepted

## Context

We need to record architectural decisions made on this project.

## Decision

We will use Architecture Decision Records (ADR), as described by Michael Nygard in this article: http://thinkrelevance.com/blog/2011/11/15/documenting-architecture-decisions

We will keep these records in the `docs/adr` directory.

## Consequences

See Michael Nygard's article, linked above.
